🛠 Team Skills Catalog

Claude Code を強化する共有スキルライブラリ

セットアップ方法

① 初回導入（~/.claude/skills 配下に配置）

mkdir -p ~/.claude/skills && cd ~/.claude/skills && git clone {{ repoUrl }}

② 最新版に更新（git pull）

cd ~/.claude/skills/{{ repoName }} && git pull
セットアップ後は Claude Code を再起動 し、チャット欄に 「/」 と入力するとスキルを呼び出せます。
